# Personal Portfolio Builder

Welcome to the **Personal Portfolio Builder** project! 🎉

This is a simple web-based tool that allows you to create your personalized portfolio (CV). Whether you're a job seeker, freelancer, or just someone who wants to showcase their skills, this tool helps you create a professional portfolio easily.

## Features

- **Enter Personal Information**: Input your personal details, including your name, contact information, education, work experience, skills, and more.
- **Edit Your Data**: After entering your data, you can return to edit and update it anytime. Your changes will be saved automatically.
- **Download Your Portfolio**: Once you're satisfied with your portfolio, you can download it in a clean, professional PDF/CV format.
- **Simple and Easy-to-Use**: No complex setup, just fill out the form and generate your portfolio in a few clicks!

## How It Works

1. **Enter Your Information**: 
   - Input personal details like your name, contact details, skills, education, experience, and projects.
   - Add extra sections like hobbies, certifications, or anything you'd like to include in your portfolio.

2. **Edit Data**: 
   - If you need to update your portfolio later, simply edit the sections with new information. This feature makes it easy to keep your portfolio up to date.

3. **Download Portfolio**: 
   - After you've finished filling out the form, click the "Download" button to generate a PDF version of your portfolio.
   - The download will include all the sections you've filled in, giving you a professional document to share with potential employers or clients.

## Technologies Used

- **HTML**: To build the structure of the portfolio web page.
- **CSS**: To style the page and ensure a clean, user-friendly design.
- **JavaScript**: To handle user input, data manipulation, and allow for editing.
- **jsPDF**: To generate a downloadable PDF file of the portfolio.

## Installation

To set up the project locally on your computer, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/AmnaMustafa1/Personal-Portfolio-Builder.git
2. Navigate to the project directory:
   ```bash
   cd Personal-Portfolio-Builder
3. Open the index.html file in your web browser to use the application locally.

## Usage
1. Open the index.html file in your web browser.
2. Fill in the fields with your personal information. Include sections like name, skills, experience, education, and any other details you want to showcase.
3. You can edit any section of the portfolio as many times as needed before finalizing your content.
4. When you're ready, click the **"Download Portfolio"** button to generate and download your personalized portfolio in PDF format.
